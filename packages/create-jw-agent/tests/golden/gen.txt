=== README.md ===
# my-gen

An agent plugin for [`jw-agent-toolkit`](https://github.com/eliascipre/jw-agent-toolkit) (Fase 41 plugin SDK).

## Install (editable)

```bash
uv sync
uv run pytest
```

## Verify discovery

```bash
uv pip install -e .
uv run jw plugins list
uv run jw plugins verify my_gen
```

## Implement your logic

Edit `src/my_gen/agent.py`. The stub returns a placeholder finding;
replace with real calls to `jw-core` / `jw-rag` / `jw-agents`.

See [cookbook recipes](https://github.com/eliascipre/jw-agent-toolkit/tree/main/docs/cookbook)
for copy-pasteable examples.

## Scope reminder

`jw-agent-toolkit` is 100% for Jehovah's Witnesses publications. Build plugins
that serve the JW community: meeting prep, apologetics, ministry tools,
specialized parsers for JW formats, etc.

=== tests/test_my_gen.py ===
"""Tests for my-gen generation provider. Deterministic, no network."""

from __future__ import annotations

from my_gen.gen import My_genGen


def test_attributes_present() -> None:
    gen = My_genGen()
    assert gen.name == "my_gen"


def test_is_available() -> None:
    assert My_genGen().is_available()


def test_generate_returns_string() -> None:
    out = My_genGen().generate("hello", max_tokens=10)
    assert isinstance(out, str)


def test_generate_respects_max_tokens() -> None:
    out = My_genGen().generate("x" * 100, max_tokens=5)
    assert isinstance(out, str)
