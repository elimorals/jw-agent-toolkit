=== README.md ===
# my-embedder

An agent plugin for [`jw-agent-toolkit`](https://github.com/eliascipre/jw-agent-toolkit) (Fase 41 plugin SDK).

## Install (editable)

```bash
uv sync
uv run pytest
```

## Verify discovery

```bash
uv pip install -e .
uv run jw plugins list
uv run jw plugins verify my_embedder
```

## Implement your logic

Edit `src/my_embedder/agent.py`. The stub returns a placeholder finding;
replace with real calls to `jw-core` / `jw-rag` / `jw-agents`.

See [cookbook recipes](https://github.com/eliascipre/jw-agent-toolkit/tree/main/docs/cookbook)
for copy-pasteable examples.

## Scope reminder

`jw-agent-toolkit` is 100% for Jehovah's Witnesses publications. Build plugins
that serve the JW community: meeting prep, apologetics, ministry tools,
specialized parsers for JW formats, etc.

=== tests/test_my_embedder.py ===
"""Tests for my-embedder embedder. Deterministic, no network, no model download."""

from __future__ import annotations

from my_embedder.embedder import My_embedderEmbedder


def test_attributes_present() -> None:
    emb = My_embedderEmbedder()
    assert emb.name
    assert emb.target in {"cpu", "gpu", "mlx"}
    assert emb.dim > 0


def test_is_available() -> None:
    assert My_embedderEmbedder().is_available()


def test_embed_returns_correct_shape() -> None:
    emb = My_embedderEmbedder()
    vecs = emb.embed(["hello", "world", "foo"])
    assert vecs.shape == (3, emb.dim)


def test_embed_handles_empty_list() -> None:
    vecs = My_embedderEmbedder().embed([])
    assert vecs.shape[0] == 0
